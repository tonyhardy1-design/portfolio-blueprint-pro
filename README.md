# Tony Hardy — Photography Portfolio

A personal portfolio website for Tony Hardy, photographer. The site showcases architecture, street, portraiture, and urban documentary work.

Built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Sections

- **Work** — Full portfolio grid across all disciplines
- **Street** — Street and documentary photography
- **Architecture** — Built environment and urban form
- **Daily Perspectives** — Everyday scenes and observed moments
- **About** — Biography and approach
- **Contact** — Get in touch

## Tech Stack

- [Vite](https://vitejs.dev/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

## Local Development

```sh
npm install
npm run dev
```
